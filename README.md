#### Browser guessing color game on React
## Run Locally

Clone the project

```bash
  git clone https://github.com/StasKobles/guessing_color_game
```

Go to the project directory =

```bash
  cd guessing_color_game
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
